Set Up Your Custom Node

Instructions can be found here

Your node must be HTTPS in order to connect to it via ClassicEtherWallet.com. You can [download the ClassicEtherWallet repo & run it locally](https://github.com/kvhnuke/etherwallet/releases/latest) to connect to your local node. Or, get free SSL certificate via [LetsEncrypt](https://letsencrypt.org/)
